ProdukAPI

ProdukAPI adalah layanan REST API sederhana untuk mengelola data Produk (Create, Read, Update, Delete). Untuk menggunakan API ini, kamu wajib memiliki API Key. Daftar akun, login, lalu generate API Key di dashboard.

Mulai Sekarang →

1. Cara Mendapatkan API Key

  1. Daftar akun di halaman Register
  2. Login di halaman Login
  3. Di Dashboard, klik tombol Generate API Key
  4. Simpan API Key tersebut, lalu sertakan di setiap request ke endpoint data via header x-api-key

2. Autentikasi

Ada dua jenis kredensial:

JenisDipakai untukCara kirim
JWT TokenAkses akun (generate/lihat API Key)Header Authorization: Bearer <token>
API KeyAkses endpoint data Produk (CRUD)Header x-api-key: <api_key>

3. Daftar Endpoint

Auth

MethodEndpointKeterangan
POST/api/auth/registerDaftar akun baru
POST/api/auth/loginLogin, mengembalikan JWT token

API Key (butuh JWT)

MethodEndpointKeterangan
POST/api/keys/generateMembuat API Key baru
GET/api/keys/my-keysMelihat semua API Key milik akun
PUT/api/keys/revoke/:idMenonaktifkan API Key

Produk (butuh API Key di header x-api-key)

MethodEndpointKeterangan
POST/api/productsTambah produk
GET/api/productsLihat semua produk
GET/api/products/:idLihat 1 produk
PUT/api/products/:idUpdate produk
DELETE/api/products/:idHapus produk

4. Contoh Request

POST /api/products
Headers: 
  Content-Type: application/json
  x-api-key: APIKEY-xxxxxxxxxxxx

Body:
{
  "nama_produk": "Keyboard Mekanik",
  "harga": 450000,
  "stok": 10,
  "deskripsi": "Keyboard gaming RGB"
}

5. Contoh Response

{
  "message": "Produk berhasil dibuat",
  "id": 1
}
